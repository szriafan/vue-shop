## 运行

``` bash
# 安装依赖
npm install

# 在localhost:8080上运行热替换服务
npm start

# 构建精简的线上版本
npm run build

# 构建线上版本并查看相关的分析报告
npm run build --report
```

想进一步了解构建工作原理, 请查看[指南](http://vuejs-templates.github.io/webpack/)和[vue-loader文档](http://vuejs.github.io/vue-loader)。
